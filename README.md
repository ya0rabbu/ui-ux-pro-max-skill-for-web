# UI UX Pro Max — Claude.ai Skill

A professional UI/UX design intelligence skill for **Claude.ai** (inspired by [ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)).

## What it does

Before writing any UI code, Claude automatically runs a **Design System Generator** that picks:

- ✅ The right UI style (67 styles — Glassmorphism, Bento Grid, Brutalism, etc.)
- ✅ Industry-matched color palette
- ✅ Typography pairing (Google Fonts)
- ✅ Layout pattern
- ✅ Effects & interactions
- ✅ Anti-patterns to avoid

Then outputs a design block before writing code:

```
┌─────────────────────────────────────────────────────────────────┐
│  DESIGN SYSTEM — Serenity Spa                                  │
├─────────────────────────────────────────────────────────────────┤
│  STYLE     Soft UI Evolution                                    │
│  PATTERN   Hero-Centric + Social Proof                          │
│  COLORS    Primary: #E8B4B8  Secondary: #A8D5BA  CTA: #D4AF37  │
│            Background: #FFF5F5  Text: #2D3436                   │
│  TYPE      Cormorant Garamond / Montserrat                      │
│  EFFECTS   Gentle transitions 200ms, soft shadows               │
│  AVOID     Neon colors, harsh animations, dark mode             │
└─────────────────────────────────────────────────────────────────┘
```

## Install

1. Download `ui-ux-pro-max.skill`
2. Go to [claude.ai](https://claude.ai) → **Settings → Skills → Upload Skill**
3. Drop the file in

## Triggers

The skill activates automatically when you ask to:

- Build / design / create any UI, page, dashboard, app, form
- "Make it look nice", "style this", "pick colors"
- "What style should I use?", "design system", "UI review"

## What's inside

| Feature | Details |
|---|---|
| UI Styles | 67 styles mapped to industries |
| Color Palettes | 10 industry-specific palettes |
| Typography | 10 font pairings (Google Fonts) |
| Layout Patterns | 7 proven patterns |
| Pre-delivery checklist | Contrast, hover states, accessibility |
| Stack support | HTML+Tailwind, React, Vue, plain CSS |

## Stacks supported

HTML + Tailwind · React · Vue / Nuxt · Angular · Plain HTML/CSS
